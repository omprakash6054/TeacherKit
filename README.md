## 🛠️ Getting Started

All tools run entirely in the browser - just open `index.html` or navigate directly to any `.html` file.

```bash
# Clone the repo
git clone https://github.com/omprakash6054/teacherkit.git

# Open in browser
open index.html
```

For the image/PDF compression tools, a local server is recommended:

```bash
npm install
```
npm run dev
